# 快速启动指南

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动服务器
```bash
npm start
```

服务器将在 `http://localhost:3000` 启动

### 3. 访问网站
打开浏览器访问: http://localhost:3000

## 测试账号

## 常见问题

### 端口被占用
如果3000端口被占用，可以修改 `server.js` 中的端口号：
```javascript
const PORT = process.env.PORT || 3000; // 改为其他端口，如3001
```

### 依赖安装失败
确保Node.js版本 >= 14.0.0：
```bash
node --version
```

### 数据文件问题
如果数据文件损坏，可以从 `data/` 目录下的示例文件重新创建。
